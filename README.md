## Complexity

time complexity of each sorting algorithms

### Selection Sort

#### Loops

Best Case

```

```

Worst Case

```
n: n-1, n-2, n-3, ., ., 1
```

#### Time (python3 and macOs\*)

Best Case

```
n: 1k, time: 0.5s

```
n: 1k, time: 0.5s
